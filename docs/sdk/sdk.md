## Enbed SDK

The Enbed SDK provides pre-built UI widgets that handle the entire insurance purchasing flow — product display, proposal form, payment, and certificate generation.

**Supported platforms:**
- **Android** — AAR library
- **iOS** — Swift + WKWebView
- **Web** — JavaScript widget
