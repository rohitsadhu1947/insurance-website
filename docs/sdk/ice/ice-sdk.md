## Overview

The ICE SDK will provide a drop-in insurance purchasing experience for **Health**, **Motor**, and other ICE-supported products directly within your web application.

---

## Planned Features

| Feature | Description |
|---------|-------------|
| **Quote Widget** | Embeddable quote comparison component |
| **Proposal Form** | Dynamic, pre-built proposal forms per insurer |
| **Payment Integration** | Built-in payment gateway handling |
| **Policy Viewer** | Download and view issued policies |
| **Multi-product** | Support for Health, Motor, and more |
