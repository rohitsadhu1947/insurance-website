## 2025

### Enbed API v1
- Full embedded insurance API — products, proposals, purchase, cancellation
- Certificate download endpoint
- Cancellation flow with refund tracking

### ICE Health API v0
- Health insurance buying journey — quote, proposal, payment
- Dynamic form fields with validation rules
- Multi-insurer quote comparison

### Vehicle Lookup API
- Vehicle registration data retrieval
- Challan/traffic violation lookup
- Available in v0, v1, v2

### Motor Insurance API
- Motor insurance buying journey
- Available in v1, v2

### Enbed SDK
- Android AAR library
- iOS WKWebView wrapper
- Web JavaScript widget
